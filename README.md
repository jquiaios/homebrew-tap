# homebrew-tap

Homebrew formulae for tools by [@jquiaios](https://github.com/jquiaios).

## Usage

```bash
brew install jquiaios/tap/<formula>
```

## Available formulae

- [`worklog`](https://github.com/jquiaios/worklog) — Fast CLI + TUI for logging work highlights and lowlights throughout the year.

  ```bash
  brew install jquiaios/tap/worklog
  ```

## How it works

Formulae in this tap are generated and updated automatically by [GoReleaser](https://goreleaser.com/) on each upstream release. No manual edits.
